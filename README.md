## fayyazazam.ca
Portfolio-ish website made from scratch.

### Some of the frameworks/libraries used:
- React
- React Bootstrap
- Webpack
- Autoprefixer
- Uglify2

### Also:
- Font Awesome icons
- Google fonts

### To run locally:
- Clone
- npm install
- npm run build-lib
- npm run build
- open index.html
